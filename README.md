### Welcome to ***Food For Zot***, an innovative website for all of your grocery shopping needs! 

Created for UC Irvine WebJam 2025 Hackathon by Eshaan Rawat, Farin Soriano, Michelle Lee, and Yoav Feigenbaum.  
Built using **Next.js** and **React** for smooth user interaction, **Flask/Python** for backend integration with a **Firestore database**, and **Google Gemini API** for recipe generation!

Run using `npm run dev`
